# [String](../README.md)

## words

`words` function returns an array of words out of input string.

```js
const result = S.words('We are more in love with desire than desired')

console.log(result)

/* Output array of words:
[
  'We',
  'are',
  'more',
  'in',
  'love',
  'with',
  'desire',
  'than',
  'desired'
]
*/
```
