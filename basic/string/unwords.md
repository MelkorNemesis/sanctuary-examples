## unwords

`unwords` function returns an array of words based on input string.

```js
const S = require('sanctuary')

const result = S.unwords([
  'We',
  'are',
  'more',
  'in',
  'love',
  'with',
  'desire',
  'than',
  'desired'
])

console.log(result) // We are more in love with desire than desired
```
