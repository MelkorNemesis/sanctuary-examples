## not

`not` returns logical `NOT` (!) of the given value.

|      | True  | False |
|------|-------|-------|
| NOT  | False | True  |

```js
const S = require('sanctuary')

console.log(S.not(true))  // false
console.log(S.not(false)) // true
```
