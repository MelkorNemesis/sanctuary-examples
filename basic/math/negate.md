# [Math](../README.md)

## negate

`negate` function returns input number multiplied by -1.

```js
const S = require('sanctuary')

console.log(S.negate(7)) // -7
console.log(S.negate(-2)) // 2
console.log(S.negate(0)) // -0
console.log(S.negate(-0)) // 0
```
